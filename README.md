SQL Library Manager
Dependencies: 
  -Sequelize 
  -Sequelize CLI 
  -Sqlite3 
  -Express 
  -Pug
In order to run this program:
  1. Download and open project files
  2. Install necessary dependencies with 'npm install'
  3. To start, type 'npm start'
